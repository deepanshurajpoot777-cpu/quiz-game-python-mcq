A console-based Python quiz game with multiple difficulty levels, scoring system, and high score tracking.

# Interactive Python Quiz Game (MCQ Based)

A command-line based quiz application built using Python.  
This project allows users to test their Python knowledge through multiple-choice questions and track their performance using a scoring and high score system.

---

## Features

- Three difficulty levels: Easy, Medium, Hard
- Multiple-choice questions (A, B, C, D)
- Random question order for each game
- Score calculation with percentage
- High score tracking system using file handling
- Game history saved in a text file
- Option to play multiple times

---

## Concepts Used

This project demonstrates the following Python concepts:

- Lists and dictionaries
- Loops and conditional statements
- Functions
- File handling (read/write operations)
- Exception handling (try-except)
- Random module
